####REAL-TIME ELECTRICITY CONSUMPTION MONITORING SYSTEM####

Software Requirements

• Arduino IDE
• ESP32 Board Package
• ThingSpeak Account
• Web Browser

Hardware Requirements

• ESP32
• ACS712 Current Sensor
• Voltage Sensor
• Wi-Fi Connection

Steps to Run the Project

1. Connect sensors with ESP32.
2. Open Arduino IDE.
3. Install ESP32 board package.
4. Upload the .ino file to ESP32.
5. Configure Wi-Fi SSID and Password.
6. Add ThingSpeak API Key in code.
7. Open serial monitor.
8. Monitor data in ThingSpeak dashboard.
